# PowerGrid Assistant

PowerGrid Assistant is a Retrieval-Augmented Generation (RAG) chatbot designed to help engineers query technical documentation related to power substation automation systems.

## Features

- Parent-child chunking
- Hybrid retrieval (BM25 + vector search)
- HyDE query expansion
- Streamlit interface

The system processes technical documents from AEGCL substation automation projects and allows users to ask natural language questions about system architecture, SCADA functionality, and technical specifications.
